Scénario Débutant #2

Après avoir brillamment trouvé le premier code, vous voici face à un nouveau défi. Cette fois-ci, vous remarquez un fichier étrange dans la machine. Son nom est une suite incompréhensible de caractères. Objectif : retrouver le nom original de ce fichier. Ce nom a été transformé à l’aide d’un algorithme de hachage (MD5). À vous de le décrypter ou de le retrouver par déduction ! Voici l’indice laissé dans un fichier .note.txt : Indice : FROMAGE Une fois le nom déchiffré, entrez-le dans la barre de recherche. Bonne chance, et n'oubliez pas de rester méthodique !

Scénario Débutant #3

Encore bravo pour avoir franchi l’étape précédente ! Ce défi vous plonge dans le monde fascinant de la stéganographie. Un fichier image .png se trouve sur la machine. Il semble anodin… mais un œil averti sait que les apparences peuvent être trompeuses. Objectif : trouver le code caché à l’intérieur de l’image. Voici ce que vous savez : Indice : Utilisez votre regard numérique… ou un outil qui sait lire entre les pixels. Petit conseil : pensez aux outils en ligne ou en local comme zsteg, steghide, ou encore la commande strings. Une fois le code trouvé, entrez-le pour valider votre progression. La chasse continue, et chaque pixel compte !
