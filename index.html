    Scénario Débutant #2

    Bravo ! Vous avez brillamment trouvé le premier code. Mais ce n'est pas fini, un nouveau défi vous attend.

    Cette fois-ci, vous remarquez un fichier étrange sur la machine. Son nom est une suite incompréhensible de caractères. Votre objectif : retrouver le nom original de ce fichier.

    Il se trouve que ce nom a été transformé à l’aide d’un algorithme de hachage. À vous de le décrypter ou de le retrouver par déduction !

    Bonne chance !
